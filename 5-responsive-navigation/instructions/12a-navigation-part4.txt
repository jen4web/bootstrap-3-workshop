Want the nav bar to go all the way across the top of the page?

1. lines 13-43 -- cut and remove outside the container

2. Wrap that row of navigation inside of a div with class of container-fluid

3. Want the nav bar fixed? Add a class of navbar-fixed-top to the nav tag. This will cut off the top part of the page, so you will need to adjust the CSS as well to take this into account.
http://getbootstrap.com/components/#navbar-fixed-top

body {
  padding-top: 70px;
}
